# falcon-ng-demo
BITS Angular JS talk and demo. 

###### Angular JS app running on top of falcon.

Installing
----

```
pip install falcon
pip install pymongo
npm install
bower install
```

Run the app locally
----

```
cd lib/
chmod +x main.py
./main.py
> Serving on 127.0.0.1:8000
```

Open your browser to <http://127.0.0.1:8000>
